ParkHere API (1.0.0)

Download OpenAPI specification:Download

API для работы с приложением поиска парковочных мест

Parking

Взаимодействие с парковочными местами

Получение списка парковочных мест

Возвращает список парковочных мест. Радиус поиска зависит от уровня подписки пользователя

Authorizations:
BearerAuth
query Parameters
lat
required
number

Широта центра области поиска.

lon
required
number

Долгота центра области поиска.

radius
required
integer
Enum: 150 1000

Радиус поиска в метрах. Максимальное значение определяется подпиской

Responses

https://api.parkhere.com/api/v1/parking

Response samples

  • 200
Content type
application/json
[
  • {
    • "parkingId": "string",
    • "lat": 0,
    • "lon": 0,
    • "address": "string",
    • "price": 0,
    • "prediction": 0
    }
]

Изменить состояние парковочного сессии

Позволяет обновить состояние парковочной сессии

Authorizations:
BearerAuth
path Parameters
sessionId
required
string

Уникальный идентификатор сессии парковки

Request Body schema: application/json
required

newState
string
Enum: "active" "paused" "completed"
